# Nitro.rar
### Script in python that removes silence parts from youtube video

## Usage

- `chmod +x nitro.py`
- `./nitro.py -l <link>`

## Flags
- `-h` help
- `-v` `--verbose` prints moments in seconds that will be cut from original video
- `-l` `--link` link to youtube video

## Dependencies
- `moviepy`
- `youtube_dl`
